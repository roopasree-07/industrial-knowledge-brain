# 🧠 Industrial Knowledge Brain
**Problem Statement 8:** AI for Industrial Knowledge Intelligence: Unified Asset & Operations Brain

An AI-powered Retrieval-Augmented Generation (RAG) assistant built for heavy industries, safety compliance officers, and field technicians. This application enables users to instantly extract verified, accurate facts from massive operational manuals and compliance PDFs, completely eliminating AI hallucinations with precise source layer mapping.

---

## 🚀 Features
* **Intelligent Document Parsing:** Instantly processes technical documentation and PDFs.
* **Context-Aware Answers:** Uses the state-of-the-art **Google Gemini API** to generate accurate answers based *only* on the uploaded file.
* **Anti-Hallucination Citations:** Every answer comes with an explicit source reference (e.g., `Source: Abstract`) ensuring 100% data accountability.
* **Lightweight UI:** Clean, fast Streamlit interface tailored for rapid field operations.

---

## 🛠️ Tech Stack
* **Language:** Python
* **Frontend Framework:** Streamlit
* **AI Model Engine:** Google Gemini (via Google AI Studio)
